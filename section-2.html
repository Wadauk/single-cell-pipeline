2 原始数据到表达矩阵

在这一章节，我们对10x genomics的下机原始数据fastq文件进行初步处理，使用Cell Ranger套件完成表达矩阵的计数。

2.1 参考数据库构建

这一步只做一次就够了，花不了很多时间。

2.1.1 单细胞参考数据库

##download ref gtf file
wget
http://cf.10xgenomics.com/supp/cell-exp/refdata-cellranger-mm10-3.0.0.tar.gz

## make custom ref gtf file
cellranger mkref --genome=GRCh38-1.2.0
--fasta=refdata-cellranger-GRCh38-1.2.0/fasta/genome.fa
--genes=GRCh38-1.2.0.gtf

2.1.2 单细胞核参考数据库

##download ref gtf file
wget
http://cf.10xgenomics.com/supp/cell-exp/refdata-cellranger-mm10-3.0.0.tar.gz

## create pre-mRNA GTF
awk 'BEGIN{FS="\t"; OFS="\t"} $3 == "transcript"{ $3="exon"; print}'
refdata-cellranger-GRCh38-1.2.0/genes/genes.gtf > GRCh38-1.2.0.premrna.gtf

## make custom ref gtf file
cellranger mkref --genome=GRCh38-1.2.0_premrna
--fasta=refdata-cellranger-GRCh38-1.2.0/fasta/genome.fa
--genes=GRCh38-1.2.0.premrna.gtf
